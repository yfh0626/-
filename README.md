# -

这是一个基于 Python/Pygame 的回合制对战项目，包含角色选择、技能选择、战斗结算、登录与可信对局验证等模块。

## 项目结构

- `main.py`：游戏入口。
- `characters/`：角色定义与角色工厂。
- `skills/`：通用技能、角色专属技能与技能工厂。
- `core/`：战斗状态、移动与对局核心逻辑。
- `login/`：登录页面、用户数据与认证逻辑。
- `crypto/`：可信对局相关的加密工具。
- `ui/`：图片页面、技能按钮等界面组件。
- `assets/`：背景、角色、技能等图片资源。

## 运行方式

1. 安装 Python 依赖：

```bash
pip install -r requirements.txt
```

2. 启动游戏：

```bash
python main.py
```

## 可信对局机制

项目通过 RSA 与 AES 组合实现“先承诺、后公开验证”的可信对局流程。

1. AI 部署阶段

   AI 生成 RSA 密钥对以及一次性的 AES 会话密钥。界面展示 RSA 公钥，以及使用 RSA 公钥加密后的 AES 密钥。

2. 对局回合阶段

   AI 出招时，使用 AES 会话密钥和随机 Nonce，以 AES-CTR 模式加密本回合操作。密文会先展示在屏幕上，之后玩家再出招。

3. 结算阶段

   对局结束后公开 RSA 私钥。玩家可以用私钥解密 AES 会话密钥，再用 AES 密钥解密历史回合密文，从而验证 AI 是否提前承诺了自己的操作。
